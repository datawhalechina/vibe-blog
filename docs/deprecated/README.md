## vibe-reviewer

vibe-reviewer 曾用于克隆 Git 仓库并评估技术教程质量。目前它不再作为 VibeBlog 的推荐功能。`REVIEWER_ENABLED` 控制导航显示及后端 API 和服务的注册或初始化；默认关闭时后端 `/reviewer` 返回 403，前端仍保留 `/reviewer` 兼容路由。

当前处理原则：

- 不再在项目 README、功能清单或主流程文档中推广。
- 现有代码和环境变量暂时保留，避免在文档调整中同时引入运行时回归。
- 仅处理必要的兼容性或安全问题，不继续扩展功能。
- 完整删除代码、路由、配置和依赖时，应提交单独 PR，并补充启动与回归测试。

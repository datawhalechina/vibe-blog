# 深度研究技能

## 适用场景
- 用户要求写深度分析、研究报告、行业洞察类文章
- 需要多角度调研和数据支撑

## 研究方法论

### Phase 1: 广度探索 (Broad Exploration)
- 使用多个搜索源进行初始搜索
- 识别主要子主题和关键角度
- 收集不同立场和观点

### Phase 2: 深度挖掘 (Deep Dive)
- 对每个子主题进行深入搜索
- 查找一手数据源（论文、官方报告）
- 收集具体案例和数据点

### Phase 3: 多样性与验证 (Diversity & Validation)
- 交叉验证关键数据
- 寻找反面观点和批评
- 确保信息来源多样性

### Phase 4: 综合检查 (Synthesis Check)
- 检查是否覆盖所有关键角度
- 验证论证逻辑链完整性
- 确保结论有充分数据支撑

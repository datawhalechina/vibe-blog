## 设计特点

### 终端美学元素
1. **窗口控制点**：红黄绿三个圆点
2. **命令提示符**：`$`, `>`, `❯`
3. **等宽字体**：JetBrains Mono
4. **代码语法色**：紫色关键字、绿色注释

### 动画效果
- 光标闪烁动画
- 卡片悬停上浮
- 箭头滑动
- 渐入动画
- 标签颜色变化

### 响应式设计
- 移动端优化布局
- 自适应网格
- 触摸友好的交互

## 技术栈
- Vue 3 Composition API
- TypeScript
- CSS Variables (Design Tokens)
- 响应式设计

# 🎨 Vibe-Blog 新配色方案

## 设计理念

采用现代化的渐变紫蓝色系，结合 Dracula 终端主题，打造专业、优雅、富有科技感的视觉体验。

---

## 🌈 主色调

### Primary - 渐变紫蓝
- **主色**: `#6366f1` (Indigo 500)
- **悬停**: `#4f46e5` (Indigo 600)
- **渐变**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- **用途**: 主要按钮、链接、强调元素

### Secondary - 粉红强调
- **主色**: `#ec4899` (Pink 500)

## 🎨 设计亮点

### 1. 渐变按钮
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
box-shadow: 0 8px 16px rgba(99, 102, 241, 0.2);
```
- 悬停时上移 + 阴影增强
- 光泽扫过动画

### 2. 卡片悬停
```css
/* 顶部渐变条从左到右展开 */
transform: translateY(-4px);
box-shadow: 多层阴影 + 彩色发光;
```

### 3. Logo 动画
```css
/* 渐变文字 + 下划线从左到右展开 */
background: linear-gradient(...);
-webkit-background-clip: text;
```

### 4. 终端粒子
- 使用 Dracula 配色
- 8 种不同颜色的代码符号
- 浮动动画，营造动态氛围

## 📊 对比度改进

### 浅色模式
- 文字对比度: **AAA 级别** (WCAG 2.1)
- 背景层次: 3 层清晰分级
- 边框可见性: 提升 20%

### 深色模式
- 文字对比度: **AA 级别** (WCAG 2.1)
- 背景更深邃: 减少眼睛疲劳
- 彩色元素更亮: 提升可见性
